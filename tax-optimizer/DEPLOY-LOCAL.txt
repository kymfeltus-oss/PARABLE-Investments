LOCAL DEPLOY — Tax Optimizer (Next.js)
=====================================

1) Open PowerShell and go to the app folder:

cd "c:\Users\kymfe\Downloads\PARABLE\tax-optimizer"

2) Install dependencies (first time or after package changes):

npm install

3) Development server (hot reload):

npm run dev

4) In your browser open:

http://localhost:3000

---

Optional — production build then run locally:

npm run build
npm run start

(Port defaults to 3000; if busy, Next.js will suggest another URL in the terminal.)
